## 🛠️ Technology Stack Used

- MongoDB with Mongoose
- Socket.IO for real-time
- JWT for authentication
- Zod for validation
- Winston for logging

- Zustand for state
- Tailwind CSS for styling
- Axios for HTTP
- Socket.IO client
- Lucide React icons

- WebSocket for real-time
- JWT token auth
- Environment-based config
